RF and Params # Calculations

Formula
---



$$channel_{out}= {( channel_{in} + (2\times padding) - kernel)\over{stride}} +1 $$

$$j_{out} = j_{in}\times stride$$

$$ rf_{out} = rf_{in} + (kernel-1)\times j_{out}$$
